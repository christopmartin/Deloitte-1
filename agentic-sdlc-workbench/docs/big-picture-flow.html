✨ marks the steps that use AI: extracting the design from a requirement, checking it for duplicates and conflicts, planning what to import from ServiceNow, and reconciling that imported data against the existing design. The other steps are people making decisions or the system assembling and packaging.

The teal-outlined steps are the ServiceNow-specific ones. A brand-new application skips them on its very first pass (nothing to link or import yet). After that first design is deployed to a ServiceNow Dev instance, the app exists there — so every later requirement runs the full ten-step loop.
