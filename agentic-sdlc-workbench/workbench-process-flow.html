AI touchpoints in this flow

  • AI Interviewer (Path A, initial + re-gather): Claude Skill conducts the PO interview and any follow-up sessions autonomously using the structured Interview Guide.
  • Whisper Transcription (Path B): Audio recordings converted to text by OpenAI Whisper before upload.
  • Claude Extraction Agent (each loop iteration): Reads the document, calls 10 typed extraction tools, self-assesses confidence, raises targeted clarification questions, and on re-runs uses the provided answers to improve accuracy. Loops until all items are staged.
  • Claude Code (Generation): Consumes the approved Workbench export package and generates ServiceNow configuration artifacts.
  • ServiceNow SDK (Deploy): Applies generated configuration to the target instance programmatically.
  • Human gates remain at: interview conduct (Path B), clarification answers, Change Packet approve/reject, baseline locking, trust dial settings, and final validation. AI never acts without a human checkpoint in the current design.
